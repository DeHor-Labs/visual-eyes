# X Thread: Visual Eyes para Claude Code 👁️🚀

1/7 O Claude Code é incrível para manipular arquivos e rodar comandos, mas para quem faz frontend, ele ainda era cego. Tentar explicar um bug visual por texto para uma IA é como descrever cores por telefone. Frustrante e lento. 🧵

2/7 Para resolver isso, criei o **Visual Eyes**. É uma skill que dá olhos ao Claude Code, permitindo que ele tire screenshots da sua aplicação, analise o layout e valide correções visualmente em tempo real.

3/7 Como funciona? O Visual Eyes usa Playwright para capturar o estado do seu localhost. O Claude recebe a imagem, entende o contexto (multimodal vision) e identifica problemas de alinhamento, cores ou espaçamento que você nem precisou descrever.

4/7 O diferencial é o loop de "Ver-Corrigir-Verificar". Você aponta um erro, o Claude tira um print, edita o código para corrigir e tira um NOVO print para confirmar que deu certo. Tudo isso sem você sair do terminal.

5/7 Precisa testar responsividade? Com um comando, o Claude gera capturas de iPhone, iPad e Desktop lado a lado. Ele mesmo analisa se o menu hambúrguer está funcionando ou se algum elemento quebrou no tablet.

6/7 Também incluí suporte a regressão visual (pixel diff). Se você refatorou um CSS global, o Visual Eyes compara o antes e depois, destacando em vermelho exatamente o que mudou na interface. Segurança total para o seu deploy.

7/7 O projeto é open source e fácil de instalar. Se você quer parar de adivinhar e começar a ver o que a IA está construindo, confira o Visual Eyes.
