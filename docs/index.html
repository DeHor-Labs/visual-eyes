How it works

Four steps. Zero friction.

Claude Code captures, analyzes, fixes, and verifies - all without leaving the chat.

📷

Capture

Playwright takes a screenshot of your localhost app at any viewport size

🔎

Analyze

Claude sees the image - layout, colors, spacing, broken elements

⚡

Fix

Claude edits the code with a clear visual target in mind

✓

Verify

Another screenshot confirms the fix looks right before moving on
