How it works

Five steps, zero friction

Just talk to Claude naturally. Visual Eyes activates automatically and handles the rest.

1

You ask Claude about your UI

No special syntax. Just speak naturally in any language.

"how does the dashboard look?"
2

Visual Eyes captures a screenshot

Playwright opens your localhost app and takes a full-quality screenshot. Desktop, mobile, or full-page.

Capturing localhost:3000...
3

Claude sees and analyzes the UI

Claude's multimodal vision reads the actual screenshot - layout, colors, typography, spacing, overflow. Everything a developer's eye catches.

"Broken grid, misaligned header, text overflow at 320px"
4

Claude fixes the bugs automatically

No descriptions needed. Claude edits your code directly, based on what it actually saw.

5

New screenshot verifies the fix

Claude takes a second screenshot and confirms the fix worked before reporting back. No more "I think it's fixed."

"Fixed. Header realigned. Want to check mobile?"
Features

Everything visual, nothing to configure

📷

Screenshot Capture

Desktop, mobile (iPhone, Android presets), full-page scroll capture, or specific routes. Any viewport, any resolution.

👁

Visual Analysis

Claude reads the actual screenshot with multimodal vision. Layout, colors, typography, spacing - no more guessing from text descriptions.

🔄

Auto-Fix Loop

Identify bug, edit code, screenshot, verify, repeat. Claude runs the loop autonomously until the visual matches your intent.

🔎

Visual Regression

Before and after pixel-level diff with highlighted changes. Red pixels show exactly what moved or changed between two states.

📱

Responsive Testing

Desktop, tablet, and mobile viewports side by side in one command. Catch responsive layout bugs before they reach production.

⚡

Zero Config

Works with any localhost app - React, Next.js, Vue, Svelte, plain HTML. No API keys, no config files, no setup wizard.

One command, then restart
